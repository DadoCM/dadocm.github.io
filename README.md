# DadoCM
Delivering Archives and Digital Objects: a Conceptual Model (DadoCM)

Available at [https://dadocm.github.io](https://dadocm.github.io)

## Server instuctions

`docker compose up serve`

## Build instructions

This site has a custom plugin for the table of contents that Github won't build itself, so it has to be built locally to `_site` before pushing to Github.

`docker compose up build`
